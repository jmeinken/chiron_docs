Transforming Table Definitions¶

You can define how to change a table def using a transformation dict. Chiron takes an existing table definition and a transformation dict, applies the transformation, and returns a new table definition. In the API, this is done using endpoint /table_def/ (POST).

add_entry¶

chiron.api.utils.table_def.apply_transformation_add_entry(user, input_table_def, transformation)¶

Add a new entry on the table definition.

Type

“add_entry”

Concept_id

(str) the permanent_id of the Concept to add

Position

(default=”append”) append|prepend new entry

Aggregation_style

(default=”stack”) stack|flatten records for field

Aggregation_method

(default=”list”) if aggregation_style is “flatten”, the method to use for aggregation

edit_entry¶

chiron.api.utils.table_def.apply_transformation_edit_entry(user, input_table_def, transformation)¶

Edit an existing entry on the table definition.

Type

“edit_entry”

Entry_id

(str) the entry_id for the table def entry to edit

Aggregation_style

(default=”stack”) stack|flatten records for field

Aggregation_method

(default=”list”) if aggregation_style is “flatten”, the method to use for aggregation

delete_entry¶

chiron.api.utils.table_def.apply_transformation_delete_entry(user, input_table_def, transformation)¶

Deletes an entry from the table def with the provided entry_id.

Type

“delete_entry”

Entry_id

The entry_id of the table def entry to remove

resort_columns¶

chiron.api.utils.table_def.apply_transformation_resort_columns(user, input_table_def, transformation)¶

Change the sort order of the table def entries (affects how columns in results table are sorted).

Type

“resort_columns”

Entry_ids

The array of entry IDs sorted in the correct order.

add_sort_entry¶

chiron.api.utils.table_def.apply_transformation_add_sort_entry(user, input_table_def, transformation)¶

Select a column to sort by (affects how rows are sorted in the results table). To reverse sort, do two add_sort_entry transformations on the same column.

Type

“add_sort_entry”

Entry_id

The entry_id of the entry to sort by.
