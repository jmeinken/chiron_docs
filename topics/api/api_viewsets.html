Parameters

  • cohort_def (list, optional) -- The current cohort definition for determining criteria set options and criteria set event options. if None then the active cohort definition will be used.

  • include_criteria_set_options (boolean, default=False) -- include options for modifying a criteria set in the cohort definition

  • include_criteria_set_event_options (boolean, default=False) -- include options for modifying a criteria set event rule in the cohort definition

  • criteria_set_entry_id (str, optional) -- If editing a criteria set in the cohort definition, the entry_id of that criteria set. Required if include_criteria_set_options is True or include_criteria_set_event_options is True

Returns

Additional data returned if include_criteria_set_options is True

CollectionViewSet._get_criteria_set_options(collection, cohort_def, entry_id)
Returns

A list of dicts representing options for manipulating criteria sets:

  • transformation - the transformation that this option applies to

  • allowed - in the context of the cohort def, can this option be applied?

  • disallowed_reason - if not allowed, why?

  • [various] - all of the other configuration values that can be set for this option

Additional data returned if include_criteria_set_event_options is True

CollectionViewSet._get_criteria_set_event_options(chironuser, collection, cohort_def, entry_id)
Returns

A list of dicts representing options for manipulating a criteria set event rule:

  • option - a unique name for this option

  • transformation - the transformation that this option applies to

  • allowed - in the context of the cohort def, can this option be applied?

  • disallowed_reason - if not allowed, why?

  • (various) - all of the other configuration values that can be set for this option
